## 📈 性能特性

### 并行执行优势

```
顺序执行: A(10ms) → B(10ms) → C(15ms) = 35ms
并行执行: [A(10ms), B(10ms)] → C(15ms) = 25ms
效率提升: 28.6%
```

## 🔧 可配置选项

### Agent 配置

- `id` - Agent 唯一标识
- `name` - Agent 名称
- `capabilities` - 支持的操作列表
- `max_concurrent_tasks` - 最大并发任务数
- `timeout_ms` - 操作超时时间
- `dependencies` - 依赖的其他 Agent

### Workflow 配置

- `id` - 工作流唯一标识
- `stages` - 工作流阶段列表
- 每个阶段配置：
  - `agents` - 参与的 Agent
  - `mode` - 执行模式（parallel/sequential）
  - `timeout_ms` - 阶段超时时间
