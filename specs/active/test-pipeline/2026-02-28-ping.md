# Feature Spec: Ping 接口

**作者**: test
**日期**: 2026-02-28
**状态**: 已批准

## 功能描述

提供一个极简的 `/api/ping` 接口，返回固定字符串，用于验证服务连通性。

## API 设计

**GET** `/api/ping`

响应 `200 OK`：
```json
{
  "message": "pong"
}
```

## 边界条件

- 无需鉴权
- 无需数据库
- 任何请求参数忽略

## 验证标准

- [ ] 请求 `/api/ping` 返回 HTTP 200
- [ ] 响应体包含 `message: "pong"`
