# Feature Spec: Memo 输入框实时字数统计

**作者**: AI Agent
**日期**: 2026-03-03
**状态**: 已批准
**关联 Issue**: #17

---

## 1. 背景

当前 Memo 输入功能缺少实时字数反馈，用户无法直观了解已输入内容长度及剩余可用字符空间。由于系统对 Memo 内容设有 10,000 字符上限，提供实时字数统计可以：
1. 帮助用户掌握输入进度
2. 在接近或超过上限时及时提醒
3. 提升用户体验，减少提交失败的情况

## 2. 用户故事

```
作为用户
我希望在输入 Memo 时能看到当前字数
以便了解还剩多少可用字符（上限 10,000），避免超限提交失败
```

## 3. 功能描述

### 交互流程
1. 用户打开 Memo 输入页面
2. 在输入框下方看到初始状态："0 / 10000"
3. 每次键入或删除内容时，字数统计实时更新
4. 根据当前字数，显示不同颜色和状态：
   - 0-9000 字：默认灰色文字
   - 9001-10000 字：橙色警告文字
   - 超过 10000 字：红色文字 + 禁用提交按钮

### 页面布局
- 字数统计文本位于输入框（textarea）正下方
- 格式："当前字数 / 10000"
- 对齐方式：右对齐或居右显示
- 颜色状态：
  - 灰色（默认）：`text-gray-500`
  - 橙色（警告）：`text-orange-500`
  - 红色（超限）：`text-red-500`

### 详细需求
- 字数统计基于 JavaScript `String.length` 属性
- 统计逻辑在客户端执行，无需调用后端 API
- 提交按钮在字数 > 10000 时禁用（`disabled` 属性）
- 禁用状态的按钮视觉上应有明显区别（如降低透明度或变灰）

## 4. 数据模型

无需修改。本功能为纯前端 UI 增强，不涉及数据库结构变更。

## 5. API 设计

无需新增 API。本功能完全在前端实现。

后端已有的验证保持不变：
- `POST /api/memos` 在 Request Body Validation 中已包含字符长度校验（≤ 10,000）
- 前端字数统计是用户体验层的辅助功能，后端验证是最终保障

## 6. 边界条件

### 输入边界
- **空输入**：显示"0 / 10000"，提交按钮可用（前提是内容非空校验在提交时处理）
- **恰好 10000 字**：显示橙色"10000 / 10000"，提交按钮可用
- **超过 10000 字**：显示红色"10001 / 10000"（或更大），提交按钮禁用

### 特殊字符处理
- Emoji 等多字节字符按 JavaScript `String.length` 计算（可能计为 2 或更多）
- 换行符 `\n` 计为 1 个字符
- 与后端验证逻辑保持一致

### 禁用提交按钮
- 字数 > 10000 时，按钮禁用
- 禁用状态下点击按钮无任何操作
- 用户删除字符使字数 ≤ 10000 后，按钮自动恢复可用

## 7. 不包含

以下功能本次不做：
- ❌ 字数统计国际化（如切换为"字"或"characters"）
- ❌ 标签字符数单独统计
- ❌ 富文本编辑器集成
- ❌ 字数历史记录或统计图表
- ❌ 剩余字数倒计时提示
- ❌ 字数预估阅读时长

## 8. 验证标准

### 功能验证
- [ ] 打开输入页面，初始显示"0 / 10000"
- [ ] 输入任意内容，字数实时更新
- [ ] 输入 9001 个字符时，文字变为橙色
- [ ] 输入 10001 个字符时，文字变为红色且提交按钮禁用
- [ ] 删除字符使总数降至 10000 以下，按钮恢复可用，颜色恢复正常
- [ ] 粘贴大段文本（超过 10000 字）时，统计和按钮状态正确响应
- [ ] Emoji 和换行符计数与后端验证一致

### 技术验证
- [ ] 字数统计使用 React state 管理
- [ ] 无 TypeScript 错误或警告
- [ ] 无 console 错误
- [ ] 代码符合项目规范（camelCase 变量名、无 `any` 类型）

### 性能验证
- [ ] 输入响应流畅，无明显卡顿（字数更新延迟 < 50ms）
- [ ] 大量文本粘贴时无 UI 冻结

---

**实现顺序建议**:
1. 在 Memo 输入组件中添加字数统计 UI（静态）
2. 添加 `useState` 跟踪字符长度
3. 绑定 `onChange` 事件实时更新字数
4. 根据字数添加条件样式（灰/橙/红）
5. 根据字数条件禁用提交按钮
6. 手工测试各种边界情况
