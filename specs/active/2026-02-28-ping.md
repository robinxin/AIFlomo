# Feature Spec: Ping 接口

**作者**: AIFlomo Team
**日期**: 2026-02-28
**状态**: 已批准

---

## 1. 背景

提供一个最简单的连通性探测接口，用于验证服务可达。

## 2. API 设计

**请求**
```
GET /api/ping
```

**响应 200**
```json
{ "message": "pong" }
```

## 3. 边界条件

- 无需认证
- 无数据库操作

## 4. 验证标准

- `GET /api/ping` 返回 200
- 响应体为 `{ "message": "pong" }`
